## ⚡ ⟨ ترسانة المميزات ⟩
> كود برمجي متقدم، صُمم ليكون خفيفاً كالشبح وقوياً كالنظام المنيع.

- ⚙️ **هيكل برمجي سلس:** بوت مصمم ليكون بسيطاً، قوياً، ومستقراً.
- 🛡️ **صيانة مستمرة:** تحديثات دائمة لضمان الأداء دون انقطاع.
- 🔘 **دعم الأزرار التفاعلية:** واجهة ديناميكية متطورة.
- 📥 **نظام التحميل الذكي:** قدرات فائقة في التعامل مع جميع التنزيلات.
- 🚀 **سرعة استجابة خاطفة:** أداء مُحسّن لمعالجة الأوامر في أجزاء من الثانية.
- 🧠 **خصائص متقدمة:** يدعم أحدث إمكانيات البوتات الاحترافية.
- 🛠️ **مرونة التعديل:** كود نظيف، مفتوح، وسهل التخصيص بالكامل.

## 💻 ⟨ بنية التشغيل والتثبيت ⟩
> **البيئة المستهدفة:** `Termux (Android)`

انسخ الأوامر التالية بالترتيب وضعها في محطة الأوامر (Terminal) لتأسيس النظام:

```bash
termux-setup-storage
pkg update -y && pkg upgrade -y
pkg install git nodejs -y
git clone [https://github.com/moreand458-eng/Escanor-bot.git](https://github.com/moreand458-eng/Escanor-bot.git)
cd Escanor-bot
npm install
npm start
